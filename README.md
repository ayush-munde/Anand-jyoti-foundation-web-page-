# React + Vite

Anand Jyoti Foundation is a modern, responsive NGO website developed using React, Vite, Tailwind CSS, Node.js, Express.js, and MongoDB. The platform highlights the foundation's mission, programs, events, gallery, volunteer opportunities, and donation initiatives while providing an intuitive user experience and an admin-friendly structure for content management. It aims to increase community engagement and support the foundation's work in empowering rural and tribal communities through education, healthcare, and sustainable livelihood programs.
